# TIMER 

#### Project made through Rocketseat's ignite course in order to practice knowledge in react.

---
## Functional requirements
- [x]  It should be able to insert a task.
- [x]  It should be able to name a task and stipulate its time.
- [x]  It should be able to start a task with a minimum value of 5 minutes and a maximum value of 60 minutes.
- [x] It should be able to access the task history.
- [x] It should be able to view all tasks and their statuses, which can be: "In progress", "Interrupted" and "Completed.

## Non-functional requirements
- [x]  It should be able to save a task in localStorage.
